О проекте:
Проект представляет собой небольшой самодельный платформер, имеющий два уровня.
в нем мы двигаемся за шпиона из игры team fortress 2, собирая багеты по пути.
при прыжке, подборе багета или попадании в лаву - проигрывается свой звук.
по достижении края первого уровня - нас телепортирует на следующий, при подходе к конце второго уровня игра заканчиватся
имеется очень маленький счетчик в нижней левой части экрана

реализована система перемещения, столкновения с обьектами, гравитация, подбор монет, проигрывание звуков и смена уровней

Калькулятор имеет размер 1280х720 пикселей
размеры шпиона и монет - средние
карты взяты уже готовые

Сайт на который я опирался при создании проекта: https://realpython.com/python-pyqt-gui-calculator/#creating-a-calculator-app-with-python-and-pyqt ](https://api.arcade.academy/en/latest/tutorials/)

Примечание: игра называется именно France, так как шпион - француз
